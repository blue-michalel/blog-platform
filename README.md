## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

- npm
- Node.js

### Installation

1. Clone the repo
2. Install NPM packages

```sh
npm install --prefix server
npm install --prefix client
```

3. Copy and adjust server .env
